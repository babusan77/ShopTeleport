# ShopTeleport
ShopTeleportは、BaburyServerのサーバプラグインです。

ショップにテレポートするやつ

## コマンド
|コマンド|説明|
----|----
/shop|設定された座標にテレポートする。(15秒後)
/shop set|テレポートする座標を設定する。
/shop c(/shop cancel)|テレポートをキャンセルする。

## 権限
|権限|使用可能コマンド|
----|----
shopteleport.use|/shop, /shop c(/shop cancel)
shopteleport.set|/shop set
